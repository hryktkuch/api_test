## ⑤工夫した点・こだわった点

- dog api(https://dog.ceo/dog-api/)を使う
-   無限の犬画像と
-   対応する犬の種類を取得できる
- deepl api(https://www.deepl.com/ja/pro-api)を使う
-   dog apiから提供される犬の名前はアルファベットでわからないので、日本語（カタカナ）に翻訳する
- ロードに結構時間がかかる（犬種が107種類ある）ので、それが終わるまで開始できないようにした

## ⑥難しかった点・次回トライしたいこと(又は機能)
